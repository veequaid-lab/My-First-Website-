# My-First-Website-
My first website — built to look like GitHub. No frameworks, just vibes and vanilla code.
# 🐙 marsh254 — Developer Portfolio

> A portfolio that looks like it was built *inside* GitHub itself.

---

## ✨ Why This Is Different

Most first websites look like Bootstrap templates.  
This one **speaks the language of GitHub** — dark theme, repo cards, contribution graph,
terminal widget, and syntax-highlighted JSON — because that's the world developers actually live in.

**The meta idea:** You're sharing this *on* GitHub, so the site *looks like* GitHub. 🔄

## 🎯 Features

| Feature | Description |
|---|---|
| 🖊️ **Typed JSON Card** | Hero section types out your profile as live JSON code |
| 📊 **Contribution Graph** | Animated GitHub-style contribution heatmap |
| 🟩 **Skill Progress Bars** | Scroll-triggered, animated bottom-border skill indicators |
| 💻 **Terminal Widget** | Bash-style contact section with real terminal aesthetics |
| 📁 **Repo Cards** | Project cards styled exactly like GitHub repository cards |
| 🔄 **Role Typewriter** | Rotating animated role titles in the hero |
| ✨ **Scroll Reveals** | Elements fade and slide in as you scroll |
| 📱 **Fully Responsive** | Works beautifully on every screen size |
| 🎨 **Zero Dependencies** | Pure HTML · CSS · Vanilla JS — no frameworks needed |

## ⚡ Get Started

### Option 1 — GitHub Pages (Recommended)
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Visit `https://yourusername.github.io/my-first-website1/`

### Option 2 — Local
```bash
git clone https://github.com/marsh254/my-first-website1.git
cd my-first-website1
open index.html   # or just double-click the file
```

## 📸 Design Inspiration

The color palette is lifted directly from GitHub's dark theme:

| Token | Hex | Use |
|---|---|---|
| Background | `#0d1117` | Page background |
| Surface | `#161b22` | Cards |
| Border | `#30363d` | All borders |
| Link | `#58a6ff` | Names, links |
| Green | `#3fb950` | Status, skills |
| Purple | `#d2a8ff` | Syntax keywords |
